Welcome to Robin's LightBox Demo!

The objective of this LightBox is to be simple, and to stay simple! In it's current state, there are only 4 options; URL, width, height and border.

LightBox is called from a javascript event. This means you can call LightBox from anything like a button through to an image. For the best results use an 'OnClick' event, but it should work with any of them, you can read more on w3schools: http://www.w3schools.com/jsref/dom_obj_event.asp

For the latest version, check out LightBox on KroFunk's GitHub: https://github.com/KroFunk/LightBox

You can connect with the author on Twitter: https://twitter.com/KroFunk.

Setup instructions

  • Place the LightBox folder in your web directory.
  • Insert the following code into your HTML just before the </body> tag. (make sure you change "src" to the relative location to your HTML)
    <script type="text/javascript" src="Path/To/LightBox/lightbox.wrapper.js"></script>
  • Setup complete! test LightBox out by copying the examples below.
  • In a production environment you should delete this example directory.

In depth look at what is going on:

It is assumed from this point on that you have a basic understanding of HTML and what an 'a' tag is. What is covered is the javascript event and what the script is doing. I will do this by colour coding the link used to open LightBox and breaking down what each part is doing.

LightBox example breakdown
<a href="javascript:void(0)" onclick="openwrapper('http://example.com',800,600,5);">Open LightBox!</a>

Explanation
Colour
NamePurpose
RedHypertext ReferenceThe value of this attribute is typically used to direct the browser to another page or another location on the current page. We want the page to stay where it is so we set javascript:void(0)
GreenHTML DOM EventThis is the type of event that will trigger a function. We want something to happen when something is clicked.
BlueJavaScript function invocation'openwrapper' is the name of a function that exists within lightbox.wrapper.js. We want to execute this function.
OrangeFunction VariablesA variable is something that can change either by design or by user input. In this case we have 4 variables; URL, Width, Height and Border. Changing these values will affect the resulting LightBox.
