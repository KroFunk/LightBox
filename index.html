Lightbox

Simple lightbox written in JavaScript

View on GitHub Download .zip Download .tar.gz

The objective of this LightBox is to be simple, and to stay simple! In it's current state, there are only 4 options; URL, width, height and border.

LightBox is called from a javascript event. This means you can call LightBox from anything like a button through to an image. For the best results use an 'OnClick' event, but it should work with any of them, you can read more on w3schools: http://www.w3schools.com/jsref/dom_obj_event.asp

For the latest version, check out LightBox on KroFunk's GitHub: https://github.com/KroFunk/LightBox
